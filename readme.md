# 前言




## 技术栈

react + redux + react-router + webpack + ES6/7+axios



## 项目运行

#### 注意：由于涉及大量的 ES6/7 等新属性，nodejs 必须是 6.0 以上版本 ，node 7 是先行版，有可能会出问题，建议使用 node 6 稳定版

```
git clone ssh://git@gitlab.gaodun.com:22122/static-sgaodun/react.git 

cd react （进入当前的项目）

npm install  (安装依赖包)

npm start (运行本地开发环境)

npm run build (打包)

另开启一个命令窗口 启动node的本地json数据代理服务

node mockserver.js (前端本地用node模拟接口进程)
```
